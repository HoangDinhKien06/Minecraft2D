
✅ SDL2 Starter Project (Code::Blocks, MinGW 64-bit)

1. Đảm bảo bạn đã tải và giải nén SDL2, SDL2_image, SDL2_mixer 64-bit.
2. Copy các .dll cần thiết (SDL2.dll, SDL2_image.dll, SDL2_mixer.dll) vào cùng thư mục với file .exe
3. Mở file SDL2_Starter.cbp bằng Code::Blocks và build/run

👉 Đường dẫn thư viện đang mặc định:
C:/sdl2/SDL2-2.28.5/
C:/sdl2/SDL2_image-2.8.5/
C:/sdl2/SDL2_mixer-2.8.0/

Bạn có thể thay đổi lại đường dẫn trong Project → Build Options nếu cần.
